# CoreX Fitness Management System

**CoreX** — это современное десктопное приложение (WPF, .NET), разработанное для комплексного управления фитнес-клубом. Система автоматизирует взаимодействие между администрацией, тренерским составом и клиентами, обеспечивая высокую скорость работы и безопасность персональных данных.

Приложение построено на многоуровневой архитектуре с четким разделением ответственности, что делает его масштабируемым и легким в поддержке.

---

## 🚀 Ключевые возможности (Features)

### 👤 Управление пользователями
- **Ролевая модель:** Разграничение прав доступа для Администраторов, Тренеров и Клиентов.
- **Безопасность:** Защищенная авторизация с использованием алгоритма хеширования **SHA256**. Реализован механизм плавной миграции для старых учетных записей (гибридная проверка пароля).
- **Администрирование:** Полный цикл CRUD-операций в панели администратора, возможность блокировки/разблокировки аккаунтов и управления штатом.

### 🏋️ Тренерский функционал и Планы
- **Управление клиентами:** Персональное закрепление атлетов за тренером, просмотр детальных карточек клиентов.
- **Планы подготовки:** Создание и редактирование индивидуальных программ тренировок и планов питания.
- **Контроль прогресса:** Интерактивное отслеживание выполнения задач клиентом в режиме реального времени через систему чек-боксов.

### 🍎 Личный кабинет и Биометрия
- **Профиль атлета:** Учет физических параметров (вес, рост, возраст) с валидацией ввода и защитой от пустых значений.
- **Автоматизация КБЖУ:** Динамический расчет суточной нормы калорий по формуле **Миффлина-Сан Жеора** при каждом изменении параметров тела.
- **Интерфейс:** Управление логином, Email и защищенная смена пароля с визуальными подсказками об ошибках.

### 🛒 Каталог и Отзывы
- **Smart Catalog:** Гибкая фильтрация абонементов по названию, цене, техническим типам (Unlimited, Morning, Group и др.) и длительности.
- **Feedback System:** Система оценки услуг (звезды) с автоматическим пересчетом рейтинга. Доступ к публикации отзывов открывается только после приобретения абонемента.

### 🎨 Интерфейс и UX
- **Dynamic Theming:** Мгновенное переключение между темами (**Оранжевая / Зеленая**) без перезапуска приложения через `ThemeManager`.
- **Localization:** Полная поддержка **русского и английского** языков с использованием динамических словарей ресурсов (`ResourceDictionary`).
- **Hotkeys:** Поддержка горячих клавиш (Ctrl+S для сохранения, Esc для отмены) для повышения эффективности работы.

## 🛠 Технологический стек (Technologies Used)

### Backend / Data Layer
- **C# / .NET Framework**
- **Entity Framework Core:** Использование подхода **Code-First** для проектирования БД.
- **MS SQL Server:** Основное хранилище данных.
- **Migrations:** Автоматическое развертывание и обновление структуры таблиц при запуске.

### Frontend / UI Layer
- **WPF (Windows Presentation Foundation):** Современный графический интерфейс.
- **XAML Styling:** Минималистичный дизайн с использованием `DynamicResource`.
- **Custom Controls:** Стилизованные таблицы (`DataGrid`), кастомные чек-боксы и диалоговые окна.

### Архитектурные паттерны (Architectural Patterns)
- **MVVM (Model-View-ViewModel):** Строгое разделение визуальной части, данных и бизнес-логики.
- **Repository + Unit of Work:** Изоляция доступа к данным и обеспечение транзакционности операций.
- **Singleton:** Реализация менеджеров управления (темы, языки, навигация).
- **Observer:** Использование интерфейса `INotifyPropertyChanged` для реактивного обновления интерфейса.
- **Strategy:** Динамическая подмена логики отображения через словари ресурсов.
